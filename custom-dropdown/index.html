Custom dropdown list

How to use:

  1. Add stylesheet in head. (You can change the style in SCSS file "custom-select.scss") <link rel="stylesheet" href="custom-select.css" />
  2. Add script in head and add "defer" attribute to it. <script src="custom-select.js" defer></script>
  3. Add class "custom-select" to the element you want to customize. <select class="custom-select"></select>

Feature:

  • Highly customizable
  • Change selection by rolling the mouse wheel on top.
  • Change selection by pressing the arrow keys.(when it's focused)
  • Open/close the dropdown list by pressing the keys: [Space], [Enter], [Esc].
